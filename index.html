Greetings! I'm Hristo Kutsarov, a dedicated web interface creator. Recently, I was accepted into the SoftUni Python Development program, where I honed my skills in Python and JavaScript software development. However, I am eager to continue learning and expanding my knowledge in the most cutting-edge techniques in the field.

To date, I have gained experience working with ORM and noSQL databases, such as PostgreSQL and MongoDB, as well as server frameworks like Django, NodeJS, and Express. Additionally, I have worked with popular front-end development frameworks, including React. Currently, I am interested in exploring TypeScript-supported frameworks such as Angular, GraphQL as a query language, cloud technologies, and solving real-life problems.

My portfolio features some of my latest projects, including a website for a local theatre developed with Django (including static pages styled with Bulma CSS), and a product management system using React with Zustand for global state management, Axios for data fetching, and Django REST for the backend. To store the application, I used Docker, and the built-in Postgres DB to manage data.
